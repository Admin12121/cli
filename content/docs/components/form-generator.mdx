## Features

- **Type Safety**: Built with TypeScript for type safety.
- **Accessibility**: Includes ARIA attributes for better accessibility.

## API Reference

### FormGenerator

| Prop        | Type   | Description                       | Default |
| ----------  | ------ | --------------------------------- | ------- |
| className   | string | The class name for the component  | ""      |
| type        | string | The type of input (e.g., text, email, password) | ""      |
| inputType   | string | The type of form element (e.g., input, select) | ""      |
| placeholder | string | Placeholder text for the input    | ""      |
| register    | any    | Function to register the input with `react-hook-form` | - |
| name        | string | Name of the input field           | ""      |
| options     | array  | Options for select inputs         | []      |
| label       | string | Label for the input field         | ""      |
| indicator   | string | Password strength indicator type  | "default" |
| required    | boolean| Whether the field is required     | false   |
| disabled    | boolean| Whether the field is disabled     | false   |

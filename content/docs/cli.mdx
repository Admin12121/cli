## Quick start: `init`

Use the `init` command to initialize configuration and dependencies for a new project.

```bash
npm jhapali-ui init
```

### Options
```bash
Usage: jhapali-ui init [options] [components...]

initialize your project and install dependencies

Arguments:
  components         the components to add or a url to the component.

Options:
  -n, --name        use define app name
  -f, --framework   use define framework i.e next or react 
  -d, --defaults    use default values i.e for next.js `src/` directory, `app-router` and `zinc`.
  -h, --help        display help for command
```

## add

The `add` command lets you easily configure and integrate popular packages into your project. Supported packages include Redux Toolkit, NextAuth, Prisma, and more.

```bash
npx jhapali-ui@latest add [packages]
```

You will be presented with a list of packages to choose from:

```bash
Which packages would you like to add? › Space to select. A to toggle all.
Enter to submit.

◯  Reduxtool kit
◯  NextAuth
◯  Prisma
more commingsoon
```

### Options
```bash
Usage: jhapali-ui add [packages]

initialize your project and install dependencies

Options:
  --prisma     use setup prisma in nextjs 
  --nextauth   use setup nextauth in nextjs
  --redux      use setup redux in you project
  -h, --help        display help for command
```
